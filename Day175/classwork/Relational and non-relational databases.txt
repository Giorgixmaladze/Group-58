ჩვენ გვაქვს ორი ტიპის მონაცემთა ბაზა:

Relational-მაკავშირებელი მონაცემთა ბაზები და non-relational-არამაკავშირებელი მონაცემთა ბაზები:
    
------------------------------------------------------------------------------------------------------------------------
Relational(მაკავშირებელი მონაცემთა ბაზები):

Relational ანუ მაკავშირებელი მონაცემთა ბაზები არიან ისეთი ბაზები,რომლებიც იყენებენ რაღაც ცხრილისმაგვარ სტრუქტურას იმისთვის,რომ შესაძლებელი იყოს მონაცემის ამოცნობა და მასზე წვდომის მოპოვება

------------------------------------------------------------------------------------------------------------------------


ასევე გვაქვს მაკავშირებელი მონაცემთა ბაზის სამართავი სისტემები(Relational Database Management System - RBDMS):


ეს არის პროგრამა რომელიც დეველოპერებს საშუალებას აძლევს, შექმნან,გაანახლონ და მართონ მაკავშირებელი მონაცემთა ბაზები.


მსგავსი პოპულარული სამართავი სისტემებისთვის შექმნილ პროგრამირების ენას ეწოდება SQL(Structured Query Language)


პოპულარული სამართავი სისტემებია:

MySQL, PostgreSQL, Oracle DB, SQL server, SQLite

------------------------------------------------------------------------------------------------------------------------



Non-relational(არამაკავშირებელი):

არამაკავშირებელი მონაცემთა ბაზები შედარებით განსხვავებულად ინახავს მონაცემებს მაკავშირებელისგან, აქ არ გვაქვს ცხრილისებრი სტრუქტურას.

არამაკავშირებელ მონაცემთა ბაზებში მონაცემი შესაძლებელია შეინახოს ერთი სტრუქტურით. მაგ:key:value წყვილები, დოკუმენტები და ა.შ.

non-relational მონაცემთა ბაზებს ხშირად უწოდებენ not only SQL ანუ არა მხოლოდ SQL 


არამაკავშირებელი მონაცემთა ბაზების უპირატესობა ისაა რომ დიდ მონაცემთან მუშაობა უფრო მარტივია ვიდრე მაკავშირებელ მონაცემთა ბაზებში. 



არამაკავშირებელი მონაცემთა ბაზის სამართავად გამოიყენება შემდეგი პროგრამები როგორიცაა MongoDB და Redis
